Excel Sheet Column Title


Description: 

Given a positive integer, return its corresponding column title as appear in an Excel sheet.

For example:

    1 -> A
    2 -> B
    3 -> C
    ...
    26 -> Z
    27 -> AA
    28 -> AB 
    ...
Example 1:

Input: 1
Output: "A"
Example 2:

Input: 28
Output: "AB"
Example 3:

Input: 701
Output: "ZY"





Solution:

建立从A到Z的字符串数组，先存(n-1)%26对应的字母在最后一位，然后之前的按n//26存储。





Code:

table="ABCDEFGHIJKLMNOPQRSTUVWXYZ"
        res=""
        while n:
            n=n-1
            res=table[n%26]+res
            n=n//26
        return res
